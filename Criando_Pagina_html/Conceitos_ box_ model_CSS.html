Fontes web são arquivos de fontes que podem ser usados em páginas da internet, permitindo que designers e desenvolvedores utilizem tipografias personalizadas sem depender das fontes instaladas no dispositivo do usuário. Essa tecnologia resolve um problema comum: antigamente, para que a fonte de um site fosse exibida corretamente, o visitante precisava ter a mesma fonte instalada em seu computador. Para garantir a compatibilidade, os desenvolvedores usavam apenas "fontes seguras para a web" (web-safe), como Arial e Times New Roman. Com as fontes web, os arquivos de fontes são baixados junto com o site, garantindo que o design seja visualizado exatamente como o pretendido, independentemente do navegador ou sistema operacional utilizado.
